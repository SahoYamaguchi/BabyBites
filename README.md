## Phase3 食材情報表示メモ

`getFoodInfo` は「注意点」シートと「調理法」シートから、食材名に一致する行を取得します。列名は `食材名` / `食材` / `材料` を食材列、`注意点` / `調理法` / `内容` / `要約` / `メモ` を本文列、`参照元` / `出典` / `日目` / `食目` を参照元列として扱います。

注意点・調理法が複数ある場合、記録入力画面で食材ボタンを選択したタイミングで箇条書き表示します。
